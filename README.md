# 説明事項

🛡️ プライバシーについて
Mini TweetPad は、ユーザーの個人情報や閲覧データを一切収集しません。
拡張機能が利用するデータは、パネルの設定やオプションなどのローカル環境に保存される情報のみであり、外部サーバーへ送信されることはありません。

本拡張機能は以下の点を保証します：

個人を特定できる情報を収集・保存・送信しません

閲覧履歴やページ内容を解析・送信しません

外部サーバーと通信する機能を持ちません

すべての処理はユーザーのブラウザ内で完結します

Mini TweetPad は、ユーザーが安心して利用できるよう、最小限の権限のみを使用し、データを外部に送らない設計になっています。
